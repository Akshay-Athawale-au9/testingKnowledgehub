The Basics

In this section we will discuss a brief walkthrough of the Product


  • After successful onboarding, You will receive a mail as shown below, click on "Login Here" Button
  • You shall get navigated to AECInspire login screen, where you can fill in Email & Password with emailed credentials
  • Once your credentials gets validated, you'll be navigated to Dashboard
  1. Your Projects: Represents all project counts you have access to.
  2. Add New Project: Button for adding a new project.
  3. Search: Helps search in your project list.
  4. Sort: Sort projects by Newest, Oldest, Last Updated, etc.
  5. Dashboard: Lists all accessible projects.
  6. Projects: Manage project operations (Create, Edit, etc.).
  7. Users: Manage user operations (Create, Edit, etc.).
  8. Q&A & Knowledge Hub: Find answers to all your queries.
  9. Notifications: Manage all notifications.
  10. Profile: Edit profile information.
  11. Project Profile: Edit or delete projects from the dashboard.
  12. Go to Project Button: Navigate inside projects.
  • Now to Create a new Project click on "Add New Project", where you will be navigated to a Project creation tab
  1. Upload Image: Here you can upload any image(png/svg/jpg/jpeg) within 1MB
  2. Project Name: Here you can enter project Name
  3. Trade: You have options to select trades (E.g.-"Electrical")
  4. Status: You can Active/Inactive your Projects
  5. Proceed: Once satisfied with data, you can simply click on "Proceed" button, to create your project.
  6. You will be able to view your created Project on Dashboard
  7. You can navigate inside the Project by clicking on the Go to project button
  • The First Screen you will see inside is your Project Dashboard
